HAESSA Component Delivery Dashboard

A full-stack web application for tracking and visualizing component delivery lifecycles across coaches in a manufacturing/assembly context.

Key Features

  • Component Management – Full CRUD with audit trail
  • Derived Status Logic – Intelligent status based on dates (Delivered, Haessa to order, CTED to place order, Overdue, Unknown, etc.)
  • Dashboard – Search, status filter, overdue alerts, sortable table
  • Gantt Chart – Timeline visualization with Today marker
  • Analytics – Coach breakdown, supplier trends, on-time performance charts
  • Security – Role-based access (admin/editor/viewer)

Technology Stack

  • Backend: Python + Flask + SQLAlchemy
  • Database: SQLite (local) / PostgreSQL (Render)
  • Frontend: Bootstrap 5 + Frappe Gantt
  • Deployment: Render.com (GitHub auto-deploy)
